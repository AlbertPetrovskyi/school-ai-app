## API Setup

To use HejChat, you need an API key from OpenRouter:

### Step 1: Sign Up on OpenRouter
First, go to the <a href="https://openrouter.ai/sign-up" target="_blank" rel="noopener noreferrer">OpenRouter</a> website and register to create an account.

<img src="./assets/rasters/openrouter_signup.png" alt="OpenRouter Sign Up" width="400">

### Step 2: Create an API Key
In your OpenRouter account settings, create a new <a href="https://openrouter.ai/settings/keys" target="_blank" rel="noopener noreferrer">API key</a>.

<img src="./assets/rasters/openrouter_key.png" alt="OpenRouter API Key" width="400">

### Step 3: Insert the API Key into HejChat
Copy the created API key and paste it into HejChat in the API section.

<img src="./assets/rasters/hejchat_api.png" alt="HejChat API Key" width="400">

## Available Models

HejChat supports the best free language models.

- Meta - Llama 3.3 70B

- Mistral - Small 3.1 24B

## How to Use

1. Insert your OpenRouter API key in the API section
2. Choose your preferred model in settings
3. Start a conversation by asking a question
4. Enjoy smooth chatting with AI!
